= Redmine Progressive Projects List Plugin

The plugin provides projects list with menus and progress bars for each project.

Plugin page and screenshots - http://www.redmine.org/plugins/progressive-projects-list

== Installation notes

Plugin's directory should be named progressive_projects_list exactly.
Other installation instructions are generic and can be found at Redmine Plugins page - http://www.redmine.org/projects/redmine/wiki/Plugins
